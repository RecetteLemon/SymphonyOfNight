= 플레이어 메뉴창 =

STR (완력 / 힘) : 공격력에 영향
CON : 체력과 기본적인 방어력에 영향
INT : 마력수치와 마법공격에 영향
LCK (행운) : 크리티컬 확률과 아이템이 나올 확률에 영향

STATUS : 현재 상태(독이나 석화등의 현재 상태, 기본은 GOOD)
LEVEL : 현재 레벨
EXP : 경험치
NEXT : 다음 레벨업에 필요한 경험치
GOLD : 현재 소유하고 있는 돈

HP : 체력 수치
MP : 마력 수치
HEART : 하트를 가진 수

ROOMS : 통과한 방의 수
BEAT : 물리친 적의 수

장비 : 무기나 아이템 등의 장비 설정
필살기 : 필살기 커맨드를 보여줌
마도기 : 고유효과를 ON/OFF 가능
시스템 : 게임의 옵션
사역마 : 사용 가능한 악마들의 상태 표시

= 장비창 =

오른손 무기(무기, 소모품)
왼손 무기(무기, 소모품)
투구
갑옷
망토
기타(장신구)
기타(장신구)

= 장비마다 영향받는 스탯들 =

무기 : 공격력에 영향을 줌
투구 : 모든 스탯에 영향을 줌
갑옷 : 공격력을 제외한 모든 스탯에 영향을 주나,
	주로 방어력에만 영향을 줌
망토 : 공격력을 제외한 모든 스탯에 영향을 줌
장신구 : 모든 스탯에 영향을 주거나,
	흡혈 등의 특수 효과가 있음
